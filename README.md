# NYC Restaurant Inspection Project

This project analyzes 2024 NYC restaurant inspection data to explore score patterns across boroughs, cuisine types, and other factors associated with restaurant inspection outcomes.

## Tools Used
- Python
- Jupyter Notebook
- Excel

## Files
- `EDA_Affix.ipynb` – exploratory data analysis
- `EDA_Prediction.ipynb` – predictive modeling
- `final_report.pdf` – final report
- `proposal.pdf` – project proposal

## Project Focus
- Borough and cuisine differences
- Temporal trends
- Factors affecting inspection scores
